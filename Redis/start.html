# 事务（multi）

# 简介

一个事务中的命令要么执行，要么都不执行

开启事务
redis 127.0.0.1:6379> MULTI 
OK

redis 127.0.0.1:6379> SET book-name "Mastering C++ in 21 days"
QUEUED

redis 127.0.0.1:6379> GET book-name
QUEUED

redis 127.0.0.1:6379> SADD tag "C++" "Programming" "Mastering Series"
QUEUED

redis 127.0.0.1:6379> SMEMBERS tag
QUEUED

退出事务
redis 127.0.0.1:6379> EXEC

# watch

可以监听一个字段，如果这个字段被修改，后续的事务是无法进行的。

如果使用watch来监听一个含有过期时间的键，该键事件到期自动删除并不会被watch监听到

# 过期时间expire（pexoire ms）

expire key time 返回值 OK

ttl key 剩余时间 （-1 永久存在 没有设置expire；-2 字段不存在）

expireat和expireat Unix时间
