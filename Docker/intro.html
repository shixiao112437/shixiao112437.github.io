# 概念

  • 镜像(image)：分片的（只读）文件系统，由Dockerfile创建
    • 独立、易扩展、更效率
  • 容器(container)：由Docker进程创建和管理的：文件系统 + 系统资源 + 网络配置 + 日志管理
    • docker是docker镜像的运行环境，所以容器的概念就比较好理解了
  • 仓库(registry)：用来远端存储docker镜像
    • 版本控制、变更管理、为持续集成与快速部署提供便利
