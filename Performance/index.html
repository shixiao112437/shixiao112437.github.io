# 浏览器的组成结构

  1. 用户界面 (地址栏,按钮,书签)
  2. 浏览器引擎 (Browser engine) 在用户界面和渲染引擎之间传送指令
  3. 渲染引擎(Rendering engine)
  4. 网络
  5. 用户界面后端
  6. JavaScript解释器 (执行js代码 如 V8引擎 js是单线程运行的)
  7. 数据存储 (Cookie)
