# 浏览器输入地址过程

  1. 对输入的网址(www.baidu.com)进行DNS域名解析,得到对应的IP地址

    • DNS解析采用的是递归擦混得方式,先去找DNS缓存

  2. 根据ip找到服务器,发起tcp的三次挥手

    • TCP是一个端到端的可靠的面向连接的协议,(发生错误时会重新传输)

  3. 建立tcp链接后发起http请求

  4. 服务器进行相应,浏览器拿到html代码

    • 解析html文件 DOM树
    • 解析css文件 渲染树
    • 边解析边渲染
    • js是单线程运行的,js有可能改变DOM解构

  5. 浏览器进行解析代码

  6. 浏览器渲染

  7. 服务器关闭tcp链接
