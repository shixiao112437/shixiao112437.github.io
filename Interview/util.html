# 防抖与节流

# debounce

概念： 在事件触发n秒后执行

    function debonce(fn,time){
        let timer = null
        return function(){
            if(timer){
                clearTimeout(timer)
                timer = null
            }
            timer = setTimeout(() => {
                fn()
            },time)
        }
    }
