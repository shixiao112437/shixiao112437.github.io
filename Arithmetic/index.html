# 栈 线性数据解构

# 理解与解释

理解: 先进先出

入栈: push 在最后

出栈:pop 删除最后一个

    class Stack {
        constructor(){
            this.stack = []
        }
        // 新增  入栈
        push(item) {
            this.stack.push(item)
        }
        // 删除 退栈 
        pop() {
            this.stack.pop(item)
        }
        // 查
        peek() {
            // 只能读取最外层的数据
           return  this.stack[this.getSize()-1]
        }
        getSize(){
           return this.stack.length
        }
        isEmpty(){
            return this.getSize() === 0
        }
    }
